HAPI 1

Krijo Bazën e të Dhënave (Firebase — Falas)

Firebase është një shërbim falas nga Google që ruan të gjitha të dhënat — pjesët, bizneset, porositë. Nuk paguan asgjë për të filluar.

1

Shko te firebase.google.com dhe kliko "Get Started". Hyr me llogarinë tënde Google (Gmail).
Hap Firebase →

2

Kliko "Create a project". Emërtoje cardeals-al. Çaktivizo Google Analytics (nuk të duhet). Kliko "Create project".

3

Pasi hapet projekti, kliko ikonën </> (Web). Emërtoje app-in cardeals-web. Kliko "Register app".

4

Do të shohësh një bllok kodi me vlera si apiKey, projectId etj. Kopjoi ato vlera — do t'i nevojitesh në hapin tjetër.

5

Në menunë e majtë, kliko Build → Authentication → Get started. Aktivizo "Email/Password" dhe ruaj.

6

Në menunë e majtë, kliko Build → Firestore Database → Create database. Zgjidh "Start in test mode". Zgjidh europe-west si lokacion. Kliko Done.

✅ Firebase është gati! Tani ke një bazë të dhënash reale falas nga Google.
HAPI 2

Vendos Kodet Firebase në Skedarët e Faqes

Duhet të bësh këtë në të dy skedarët: cardeals-al.html dhe cardeals-backoffice.html

1

Hap skedarin cardeals-al.html me Notepad (kliko me të djathtën → "Open with" → Notepad).

2

Shtyp Ctrl+H (Gjej dhe Zëvendëso). Gjej tekstin YOUR_API_KEY dhe zëvendësoje me apiKey-n tënd nga Firebase.

3

Bëj të njëjtën gjë për të gjitha vlerat e tjera:

YOUR_API_KEY → vlera apiKey nga Firebase
YOUR_PROJECT_ID → vlera projectId nga Firebase
YOUR_SENDER_ID → vlera messagingSenderId nga Firebase
YOUR_APP_ID → vlera appId nga Firebase
4

Ruaj skedarin (Ctrl+S). Bëj të njëjtën gjë edhe për cardeals-backoffice.html.

💡 Kujdes: Çdo vlerë duhet zëvendësuar saktë, pa hapësira shtesë. Nëse bën gabim, Firebase nuk do të funksionojë.
HAPI 3

Konfiguro PayPal për Pagesa Reale

PayPal të lejon të pranosh pagesa online. Është falas të hapësh llogari — paguan vetëm 3.5% kur merr para.

1

Shko te paypal.com/al dhe krijo llogari Business (jo personal).
Hap PayPal →

2

Pasi të verifikosh llogarinë, shko te developer.paypal.com. Hyr me llogarinë tuaj PayPal.

3

Kliko "Apps & Credentials" → "Create App". Emërtoje Cardeals. Kopjo Client ID.

4

Në skedarin cardeals-al.html, gjej tekstin YOUR_PAYPAL_CLIENT_ID dhe zëvendësoje me Client ID-n tënd.

✅ Tani blerësit mund të paguajnë direkt me PayPal ose kartë krediti!
HAPI 4

Publiko Faqen Online (Netlify — Falas)

Netlify është një shërbim falas që publikon faqen tënde online në internet. Nuk duhet të dini asgjë teknikë.

1

Shko te netlify.com dhe krijo llogari falas.
Hap Netlify →

2

Pasi hyn, shiko pjesën ku shkruan "Deploy manually" ose "Drag and drop".

3

Hap dosjen ku ke skedarët e Cardeals.al. Zgjedh të tre skedarët dhe tërhiqji (drag) direkt në faqen e Netlify.

4

Netlify do të krijojë automatikisht një adresë si random-name.netlify.app. Faqja jote është live!

5

(Opsionale) Nëse ke blerë domenin cardeals.al, shko te Domain settings në Netlify dhe shto domenin tënd.

💡 Netlify falas lejon deri 100GB trafik në muaj — shumë më tepër se sa do nevojitesh në fillim.
✅ Urime! Faqja jote është live në internet dhe funksionon plotësisht!
HAPI 5

Siguria e Bazës së të Dhënave

Kjo hap është i rëndësishëm për sigurinë. Duhet bërë pasi faqja të jetë live.

1

Kthehu te Firebase Console → Firestore Database → Rules.

2

Zëvendëso rregullat ekzistuese me këto:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Të gjithë mund të lexojnë pjesët aktive
    match /parts/{partId} {
      allow read: if resource.data.status == 'active';
      allow write: if request.auth != null &&
        request.auth.uid == resource.data.uid;
      allow create: if request.auth != null;
    }
    // Bizneset shohin vetëm profilet e tyre
    match /businesses/{uid} {
      allow read, write: if request.auth.uid == uid;
    }
    // Porositë — vetëm shitësi i tyre
    match /orders/{orderId} {
      allow read: if request.auth.uid == resource.data.sellerUid;
      allow create: if true;
    }
  }
}
3

Kliko "Publish". Rregullat hyjnë në fuqi menjëherë.

✅ Baza e të dhënave është e sigurt tani!
